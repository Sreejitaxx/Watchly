# Watchly

AI-powered movie recommendation web!



This project is an **AI-based movie recommendation web application** that allows users to discover movies using **natural language prompts**. Instead of manually selecting filters, users can simply describe the type of movie they want (for example: *“a sci-fi movie with space adventure”* or *“a romantic comedy set in college”*), and the system uses AI to generate relevant recommendations.

The application integrates the **Gemini API** to process user prompts and understand the intent behind the request. Based on the prompt, the AI suggests **five suitable movies**. The system then fetches detailed movie information from **The Movie Database (TMDB)**, including posters, ratings, language, runtime, and release date.

To provide secure access, the application uses **Firebase Authentication** for user authentication. Users can **sign up, log in, and log out**, ensuring that only authenticated users can access the recommendation features.

Movies are displayed in a **poster-based interface**, making the application visually engaging and easy to explore. Users can click on any movie to view detailed information such as title, rating, language, runtime, release date, and description.

The frontend is built using **React.js and Tailwind CSS**, creating a **responsive and modern UI** for smooth navigation and user interaction.

## 🚀 FEATURES
